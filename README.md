## 🎯 Key Concepts Learned

### Grid vs Flexbox

| Grid                  | Flexbox                |
| --------------------- | ---------------------- |
| Two-dimensional       | One-dimensional        |
| Rows and Columns      | Row OR Column          |
| Complex layouts       | Simple layouts         |
| Dashboards, Galleries | Navigation bars, Cards |

### Bootstrap Grid

Bootstrap uses a 12-column layout system.

Example:

```html
<div class="row">
  <div class="col-4">Column 1</div>
  <div class="col-8">Column 2</div>
</div>
```
